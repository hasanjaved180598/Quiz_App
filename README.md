# 🧠 Quiz App

A fun and interactive quiz app built with Flutter. Test your knowledge, track your score, and challenge yourself!

---

## 🎬 Demo
https://github.com/your-username/quiz_app/blob/main/assets/demo.mp4
---

## ✨ Features

- ❓ Multiple choice questions
- 📊 Score tracking throughout the quiz
- 🏁 Results summary screen at the end
- 🔄 Restart and play again
- 🔀 Answers shuffled on every restart for a fresh challenge

---

## 🏗️ Project Structure

```
lib/
├── main.dart                      # App entry point
├── quiz.dart                      # Core quiz logic
├── assets/
│   └── images/
│       └── quiz-logo.png          # App logo
├── data/
│   └── questions.dart             # Quiz questions data
├── models/
│   └── quiz_question.dart         # Quiz question model
├── answer_button.dart             # Answer option button widget
├── question_screen.dart           # Question & answer screen
├── questions_summary.dart         # Summary of all answers
├── results_screen.dart            # Final score screen
└── start_screen.dart              # Welcome / start screen
```
